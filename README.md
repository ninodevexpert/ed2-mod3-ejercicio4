# CV Digital

Web estática moderna y responsive para presentar un currículum vitae digital profesional.

## Características

- Diseño moderno y profesional
- Totalmente responsive (adaptado a todos los dispositivos)
- Animaciones CSS suaves y atractivas
- Navegación fluida con scroll suave
- Secciones: Inicio, Sobre Mí, Experiencia, Educación, Habilidades y Contacto
- Formulario de contacto interactivo
- Efectos visuales avanzados (parallax, animaciones al hacer scroll)

## Estructura del Proyecto

```
/
├── index.html      # Página principal HTML
├── styles.css      # Estilos CSS con animaciones
├── script.js       # JavaScript para interactividad
├── res/
│   ├── cv.md       # Datos del CV (Markdown)
│   └── photo.png   # Foto de perfil profesional
└── README.md       # Este archivo
```

## Uso

1. Abre `index.html` en tu navegador
2. Personaliza el contenido editando `index.html` y `res/cv.md`
3. Reemplaza `res/photo.png` con tu propia foto de perfil

## Personalización

Para personalizar el CV:

1. Edita `index.html` y reemplaza el contenido de ejemplo con tus datos reales
2. Actualiza `res/cv.md` con tu información profesional
3. Reemplaza `res/photo.png` con tu foto de perfil
4. Modifica los colores en `styles.css` cambiando las variables CSS en `:root`

## Tecnologías Utilizadas

- HTML5
- CSS3 (con animaciones y variables CSS)
- JavaScript (Vanilla JS)
- Diseño responsive con CSS Grid y Flexbox
